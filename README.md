# Dice Roller Game

A simple GUI Dice Roller Game built using Python and Tkinter.

## Features
- Random dice roll (1-6)
- Roll counter
- Simple graphical interface
- Exit button

## Enhanced Features
- Select 1, 2, or 3 dice
- Show the sum of all dice
- Show roll history of previous 5 rolls
- Reset button
- Statistics (average, highest, lowest)

## Technologies Used
- Python
- Tkinter
